kornia.geometry.homography
==========================

.. meta::
   :name: description
   :content: "The kornia.geometry.homography module provides essential tools for manipulating and working with homographies, which describe the transformation between two images of the same scene from different viewpoints. The module includes a variety of functions for computing, applying, and manipulating homographies, making it useful for tasks such as image stitching, object tracking, and perspective warping. The module also offers an interactive demo to explore homography warping in real time."
